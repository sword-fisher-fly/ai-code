# 中国象棋 - 人机对战游戏 🎮

一个使用React开发的完整中国象棋游戏，支持与AI进行智能对战。

## 🎯 游戏特色

- **完整规则实现**: 严格遵循中国象棋规则
- **智能AI对手**: 基于Minimax算法的AI，带Alpha-Beta剪枝优化
- **古朴界面**: 传统中式风格，黄色棋盘
- **计时系统**: 每方限时5分钟
- **实时对战**: 流畅的玩家交互体验

## 🎮 游戏规则

### 棋子系统
- **帅/将**: 只能在九宫格内移动，每次只能走直线一格
- **仕/士**: 只能在九宫格内斜向移动
- **相/象**: 只能在己方半场斜向移动两步，有"塞象眼"限制
- **马**: 走"日"字，有"蹩马腿"限制
- **车**: 横竖直线走，不能越子
- **炮**: 平时走法同车，吃子时必须隔一个棋子
- **兵/卒**: 过了河可以左右移动，未过河只能向前一步

### 胜负判定
- **将死**: 对方的帅/将无法逃脱被攻击
- **超时**: 计时耗尽自动判负

## 🎲 AI算法

使用**Minimax算法**配合**Alpha-Beta剪枝**：

- **搜索深度**: 3层
- **评估函数**: 考虑棋子价值、位置、灵活性
- **优化策略**: Alpha-Beta剪枝减少50%+计算量
- **移动排序**: 启发式排序提升剪枝效率

## 🎯 游戏策略

### 对玩家建议
1. **开局**: 快速出子，控制中心
2. **中局**: 协调子力，攻击弱点
3. **残局**: 精确计算，追求将杀

### 观察AI
- AI思考时间显示其计算复杂度
- 优先攻击高价值目标
- 注重子力协调和保护
